# Advanced Statistics Superstore Analysis

## Project Overview

This project applies advanced statistical techniques to analyze the Superstore Sales dataset.

The objective is to understand sales performance, profitability, customer behavior, and discount patterns using descriptive statistics and inferential statistics.

The project focuses on converting statistical analysis into meaningful business insights and strategic recommendations.

---

## Objectives

- Analyze sales and profit performance
- Study the impact of discounts on profitability
- Compare customer segments
- Compare regional performance
- Perform hypothesis testing
- Generate business insights using statistical evidence
- Recommend strategies to improve business performance

---

## Dataset

The project uses the Sample Superstore dataset.

The cleaned dataset from the Exploratory Data Analysis project is used as the input for this analysis.

---

## Statistical Techniques Used

### Descriptive Statistics

- Mean
- Median
- Mode
- Standard Deviation
- Variance
- Range
- Quartiles
- Skewness
- Kurtosis

### Inferential Statistics

- Confidence Interval
- Sampling
- Independent T-Test
- One-Way ANOVA
- Chi-Square Test
- Correlation Analysis

---

## Business Questions

The analysis answers questions such as:

- Which regions perform better?
- Which customer segment generates higher profit?
- Does discount affect profitability?
- Are customer segments associated with discount levels?
- Does average profit differ across regions?
- What business strategies improve profitability?
